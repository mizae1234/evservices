## การเข้าสู่ระบบ

### ขั้นตอนการ Login
1. เปิดเว็บไซต์ระบบ EV Services
2. กรอก **อีเมล** และ **รหัสผ่าน** ที่ได้รับจากผู้ดูแลระบบ
3. กดปุ่ม **"เข้าสู่ระบบ"**
4. ระบบจะนำไปยังหน้า Dashboard ตามสิทธิ์ผู้ใช้

> **หมายเหตุ:** หากลืมรหัสผ่าน กรุณาติดต่อผู้ดูแลระบบ

## คู่มือสำหรับศูนย์บริการ (สาขา)

### 1. หน้า Dashboard
**เข้าถึงได้ที่:** เมนู "Dashboard"

**ฟังก์ชันหลัก:**
- แสดงสรุปจำนวนเคลมทั้งหมดของสาขา
- แสดงสถานะเคลม: แบบร่าง, รออนุมัติ, อนุมัติแล้ว, ปฏิเสธ, ขอข้อมูลเพิ่ม
- รายการเคลมล่าสุด 5 รายการ
- ปุ่มดาวน์โหลด PDF สำหรับเคลมที่อนุมัติแล้ว

---

### 2. รายการเคลม
**เข้าถึงได้ที่:** เมนู "รายการเคลม"

**ฟังก์ชันหลัก:**
- แสดงรายการเคลมทั้งหมดของสาขา
- ค้นหาด้วย: เลขเคลม, ชื่อลูกค้า, ทะเบียนรถ
- กรองตามสถานะ
- กรองตามช่วงวันที่ (default: 7 วันย้อนหลัง)

**การดูรายละเอียดเคลม:**
1. คลิกที่ปุ่ม "ดู" ในคอลัมน์จัดการ
2. ดูรายละเอียดเคลม, สถานะ, ประวัติการดำเนินการ
3. ดาวน์โหลด PDF เอกสารอนุมัติ (สำหรับเคลมที่อนุมัติแล้ว)

---

### 3. สร้างเคลมใหม่
**เข้าถึงได้ที่:** เมนู "สร้างเคลมใหม่" หรือปุ่ม "+ สร้างเคลมใหม่"

**ขั้นตอน:**

#### ขั้นที่ 1: กรอกข้อมูลลูกค้า
- **ชื่อลูกค้า** (จำเป็น)

#### ขั้นที่ 2: ข้อมูลรถยนต์
- **รุ่นรถ** - เลือกจาก dropdown (จำเป็น)
- **ทะเบียนรถ** - กรอกติดกันไม่มีช่องว่าง เช่น "กท1234" (จำเป็น)
- **VIN Number** - เลขตัวถัง (ถ้ามี)

#### ขั้นที่ 3: ข้อมูลระยะทาง
- ติ๊ก "ตรวจเช็คระยะทาง" หากเป็นเคลมตรวจเช็คตามระยะ
- เลือกระยะ: 5,000 / 20,000 / 30,000 / 40,000 กม.
- กรอกไมล์ล่าสุด

#### ขั้นที่ 4: รายละเอียดเคลม
- **รายละเอียด** - อธิบายปัญหาหรือบริการ
- **จำนวนเงิน (บาท)** (จำเป็น)

#### ขั้นที่ 5: แนบเอกสาร
- กดปุ่ม "เลือกไฟล์" เพื่ออัพโหลดเอกสารประกอบ
- รองรับไฟล์: รูปภาพ, PDF, เอกสาร
- สามารถแนบได้หลายไฟล์

#### ขั้นที่ 6: บันทึก/ส่งอนุมัติ
- **"บันทึก"** - บันทึกเป็นแบบร่าง สามารถแก้ไขได้ภายหลัง
- **"บันทึกและส่งอนุมัติ"** - ส่งให้ Admin ตรวจสอบ

---

### 4. แก้ไขเคลม
**สามารถแก้ไขได้เมื่อ:**
- สถานะ "แบบร่าง"
- สถานะ "ขอข้อมูลเพิ่ม" (Admin ส่งกลับให้แก้ไข)

**ขั้นตอน:**
1. ไปที่รายการเคลม > กดดู > กดปุ่ม "แก้ไข"
2. แก้ไขข้อมูลตามต้องการ
3. กด "บันทึก" หรือ "บันทึกและส่งอนุมัติ"

---

### 5. ดาวน์โหลดเอกสารอนุมัติ (PDF)
**สำหรับเคลมที่อนุมัติแล้ว:**
1. ไปที่หน้ารายละเอียดเคลม
2. กดปุ่ม "ดาวน์โหลดเอกสารอนุมัติ"
3. ไฟล์ PDF จะเปิดในแท็บใหม่ สามารถพิมพ์หรือบันทึกได้

---

## คู่มือสำหรับผู้ดูแลระบบ (Admin)

### 1. Admin Dashboard
**เข้าถึงได้ที่:** เมนู "Admin Dashboard"

**ฟังก์ชันหลัก:**
- สรุปภาพรวมเคลมทั้งระบบ
- จำนวนเคลมรอดำเนินการ
- เคลมล่าสุดที่ต้องตรวจสอบ

---

### 2. อนุมัติเคลม
**เข้าถึงได้ที่:** เมนู "อนุมัติเคลม"

**ฟังก์ชันหลัก:**
- แสดงรายการเคลมทั้งหมดจากทุกสาขา
- ค้นหาด้วย: เลขเคลม, ชื่อลูกค้า, ทะเบียนรถ
- กรองตามสถานะ / ช่วงวันที่
- เคลม "รออนุมัติ" จะแสดงพื้นหลังสีเหลือง

**การตรวจสอบและอนุมัติ:**
1. คลิก "ตรวจสอบ" ที่เคลมที่ต้องการ
2. ตรวจสอบข้อมูล: ลูกค้า, รถยนต์, รายละเอียด, เอกสารแนบ
3. เลือกดำเนินการ:
   - **"อนุมัติ"** - อนุมัติเคลม (ระบุหมายเหตุได้)
   - **"ปฏิเสธ"** - ปฏิเสธเคลม (ต้องระบุเหตุผล)
   - **"ขอข้อมูลเพิ่ม"** - ส่งกลับให้สาขาแก้ไข (ต้องระบุข้อมูลที่ต้องการ)
4. กดยืนยัน

---

### 3. ประวัติเคลม
**เข้าถึงได้ที่:** เมนู "ประวัติเคลม"

**แสดงข้อมูล:**
- เคลมที่ดำเนินการเสร็จแล้ว (อนุมัติ/ปฏิเสธ)
- กรองตามช่วงวันที่
- วันที่อนุมัติ/ปฏิเสธ
- หมายเหตุการอนุมัติ

---

### 4. รายงาน
**เข้าถึงได้ที่:** เมนู "รายงาน"

**ฟังก์ชันหลัก:**
- ดูข้อมูลเคลมในรูปแบบตาราง
- กรองตามสถานะ (default: อนุมัติแล้ว)
- กรองตามช่วงวันที่ (default: 7 วันย้อนหลัง)
- **Export Excel** - ดาวน์โหลดข้อมูลเป็นไฟล์ .xlsx

**ข้อมูลในรายงาน:**
| คอลัมน์ | คำอธิบาย |
|---------|----------|
| เลขเคลม | หมายเลขเอกสารเคลม |
| วันที่ | วันที่สร้างเคลม |
| รายละเอียด | รายละเอียดการเคลม |
| จำนวนเงิน | มูลค่าเคลม (บาท) |
| สาขา | ศูนย์บริการที่ส่งเคลม |
| ระยะทาง | ระยะเช็คระยะ |
| รุ่นรถ | รุ่นรถยนต์ |
| ทะเบียน | ทะเบียนรถ |
| ลูกค้า | ชื่อลูกค้า |
| ไมล์ล่าสุด | เลขไมล์ล่าสุด |
| VIN | เลขตัวถัง |

**การ Export Excel:**
1. ตั้งค่า filter ตามต้องการ
2. กดปุ่ม "ค้นหา" เพื่อแสดงข้อมูล
3. กดปุ่ม "Export Excel"
4. ไฟล์จะถูกดาวน์โหลดอัตโนมัติ

---

### 5. พิมพ์เอกสารอนุมัติ
**สำหรับเคลมที่อนุมัติแล้ว:**
1. ไปที่หน้ารายละเอียดเคลม
2. กดปุ่ม "พิมพ์เอกสารอนุมัติ"
3. เอกสาร PDF จะเปิดในแท็บใหม่
4. สามารถพิมพ์หรือบันทึกได้

---

## สถานะเคลม

| สถานะ | คำอธิบาย | สี |
|-------|----------|-----|
| แบบร่าง | เคลมที่ยังไม่ได้ส่งอนุมัติ | เทา |
| รออนุมัติ | รอ Admin ตรวจสอบ | เหลือง |
| อนุมัติแล้ว | อนุมัติเรียบร้อย | เขียว |
| ปฏิเสธ | ไม่อนุมัติ | แดง |
| ขอข้อมูลเพิ่ม | Admin ต้องการข้อมูลเพิ่มเติม | ส้ม |

## การแก้ปัญหาเบื้องต้น

### ไม่สามารถ Login ได้
- ตรวจสอบอีเมลและรหัสผ่านให้ถูกต้อง
- ติดต่อผู้ดูแลระบบ

### ไม่สามารถส่งเคลมได้
- ตรวจสอบว่ากรอกข้อมูลจำเป็นครบถ้วน (ชื่อลูกค้า, รุ่นรถ, ทะเบียน, จำนวนเงิน)
- ทะเบียนรถต้องกรอกติดกัน ไม่มีช่องว่าง

### ไม่เห็นเคลมในรายการ
- ตรวจสอบช่วงวันที่ที่เลือก
- ตรวจสอบสถานะที่เลือก

หากพบปัญหาในการใช้งาน กรุณาติดต่อผู้ดูแลระบบ
